# Contributing

Do you want to contribute? We'd love that.

Our goal is to gather useful projects related to Warfarin therapy, calculation, pharmacogenomics, or clinical management.

## TL;DR

- Accepted: **All projects are welcome!** Work in this field is rare, so we gladly accept any contributions.
- Template: `- [ACCOUNT/REPO](https://github.com/ACCOUNT/REPO) - DESCRIPTION` (or `[Project Name](URL)` if not on GitHub)
- Sort: alphabetical

## Quality Standards

To ensure this list remains high-quality, please adhere to the following guidelines:

- **Relevance:** Projects must be directly related to Warfarin (Coumadin) therapy, calculation, pharmacogenomics, or clinical management.
- **Open Source:** We prioritize open-source projects. Closed-source commercial tools may be considered if they offer significant free utility.
- **Documentation:** The linked project must have a clear README (preferably in English) explaining what it does and how to run it.

## How?

The easiest way is to go to the `README.md` file in this repository and click on the "pen" icon in the upper right corner. Make the changes to the file and follow the instructions to create a pull request.

If you want to add an entry to the `README.md` please consider this:

- **Use the correct format:** Please use the template `- [ACCOUNT/REPO](https://github.com/ACCOUNT/REPO) - DESCRIPTION`. If the project is not hosted on GitHub, use `- [Project Name](URL) - DESCRIPTION`.
- **Alphabetical ordering:** Please pay attention to the alphabetical ordering within the appropriate category.

## Removing projects

We don't remove projects unless they are outright broken or pronounced deprecated by another project or by its author.
