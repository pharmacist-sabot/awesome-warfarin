## ðŸŒ Web Applications

*A collection of web-based tools and dashboards for calculating or simulating warfarin dosage.*

- **[Vue-Quick-Warfarin-V2](https://github.com/Napatchouli/Vue-Quick-Warfarin-V2)**
  > Vue.js Warfarin Dosing App for Clinicians.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/Napatchouli/Vue-Quick-Warfarin-V2?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/Napatchouli/Vue-Quick-Warfarin-V2)
  ![GitHub top language](https://img.shields.io/github/languages/top/Napatchouli/Vue-Quick-Warfarin-V2)

- **[warfarin6](https://github.com/penpharmacy/warfarin6)**
  > PWA tool for pharmacists to verify daily pill regimens. Calculates total weekly dose and determines exact 2mg/3mg pill count for dispensing.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/penpharmacy/warfarin6?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/penpharmacy/warfarin6)
  ![GitHub top language](https://img.shields.io/github/languages/top/penpharmacy/warfarin6)

- **[warfarin-app](https://github.com/pharmacist-sabot/warfarin-app)**
  > A fast web-based regimen planner using Vue.js and Rust WebAssembly for intelligent dosing schedules.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/pharmacist-sabot/warfarin-app?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/pharmacist-sabot/warfarin-app)
  ![GitHub top language](https://img.shields.io/github/languages/top/pharmacist-sabot/warfarin-app)

- **[warfarin-calculator](https://github.com/Paond55/warfarin-calculator)**
  > A guideline-based web tool for clinicians to calculate warfarin dose adjustments based on INR and bleeding symptoms. It suggests new weekly doses and generates multiple daily pill regimens. Built as a PWA for offline use.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/Paond55/warfarin-calculator?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/Paond55/warfarin-calculator)
  ![GitHub top language](https://img.shields.io/github/languages/top/Paond55/warfarin-calculator)

- **[Warfarin-Dose-Calculator](https://github.com/prajwal-17/warfarin-dose-calculator)**
  > React + TypeScript app for anticoagulant dosing, supporting Warfarin (initiation & INR-based maintenance) and Acitrom.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/prajwal-17/warfarin-dose-calculator?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/prajwal-17/warfarin-dose-calculator)
  ![GitHub top language](https://img.shields.io/github/languages/top/prajwal-17/warfarin-dose-calculator)

- **[Warfarin-Dosage-Calculator](https://github.com/pzweuj/warfarin-dosage-calculator)**
  > A modern Next.js web application that implements multiple established pharmacogenomic algorithms for predicting initial warfarin dosage. Features a clean UI and multi-language support.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/pzweuj/warfarin-dosage-calculator?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/pzweuj/warfarin-dosage-calculator)
  ![GitHub top language](https://img.shields.io/github/languages/top/pzweuj/warfarin-dosage-calculator)
  
- **[warfarin-planner](https://github.com/warfarin-net/warfarin-planner)**
  > Warfarin dose-to-pill PWA with smart, simple regimen sorting and dispensing calculations.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/warfarin-net/warfarin-planner?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/warfarin-net/warfarin-planner)

## ðŸ§  Machine Learning & Research

*Projects using ML, data analysis, and statistical modeling to predict optimal warfarin dosage.*

- **[personalized-warfarin](https://github.com/cybergeist0/personalized-warfarin)**
  > A Jupyter Notebook project that predicts personalized warfarin doses using both clinical and genetic data from the PharmGKB dataset.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/cybergeist0/personalized-warfarin?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/cybergeist0/personalized-warfarin)
  ![GitHub top language](https://img.shields.io/github/languages/top/cybergeist0/personalized-warfarin)

- **[pgx-warfarin-ml-portfolio](https://github.com/Uche-Stephen/pgx-warfarin-ml-portfolio)**
  > A scikit-learn pipeline for personalized warfarin dosing, including explainability (SHAP) and a Streamlit web app.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/Uche-Stephen/pgx-warfarin-ml-portfolio?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/Uche-Stephen/pgx-warfarin-ml-portfolio)
  ![GitHub top language](https://img.shields.io/github/languages/top/Uche-Stephen/pgx-warfarin-ml-portfolio)

- **[warfit-learn](https://github.com/gianlucatruda/warfit-learn)**
  > A machine learning project using Python and scikit-learn to predict warfarin dosage, including a Jupyter Notebook for data exploration and model training.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/gianlucatruda/warfit-learn?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/gianlucatruda/warfit-learn)
  ![GitHub top language](https://img.shields.io/github/languages/top/gianlucatruda/warfit-learn)

- **[warfarin-machinelearning](https://github.com/karneslab/warfarin-machinelearning)**
  > A Python project comparing several machine learning models to predict stable warfarin doses from clinical and pharmacogenomic data.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/karneslab/warfarin-machinelearning?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/karneslab/warfarin-machinelearning)
  ![GitHub top language](https://img.shields.io/github/languages/top/karneslab/warfarin-machinelearning)

- **[Warfarin_Dose_Prediction](https://github.com/farahyusri12/warfarin_dose_prediction)**
  > A Google Colab notebook that builds a linear regression model to predict warfarin dose from the IWPC dataset, with a final PDF report.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/farahyusri12/warfarin_dose_prediction?style=social)
  ![GitHub last commit](https://img.shields.io/github/last-commit/farahyusri12/warfarin_dose_prediction)
  ![GitHub top language](https://img.shields.io/github/languages/top/farahyusri12/warfarin_dose_prediction)

- **[warfarin-dosing-ml-tool-](https://github.com/ifthekarhussain/warfarin-dosing-ml-tool-)**
  > An end-to-end ML pipeline in a Jupyter Notebook using the IWPC dataset, culminating in an interactive prediction tool built with Gradio.
  <br>
  ![GitHub stars](https://img.shields.io/github/stars/ifthekarhussain/warfarin-dosing-ml-tool%2D?style=social)
