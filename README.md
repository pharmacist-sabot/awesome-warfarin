Warfarin is a widely used anticoagulant prescribed to millions worldwide. Due to factors like genetics, diet, and drug interactions, dosing can be challenging. This list aims to aggregate tools to improve dosing accuracy, patient management, and pharmacogenomics research.

## 🌐 Web Applications

Web-based tools and dashboards for calculating or simulating warfarin dosage. Ideal for clinicians and pharmacists.

- [Vue-Quick-Warfarin-V2](https://github.com/Napatchouli/Vue-Quick-Warfarin-V2) - Vue.js dosing app for clinicians to calculate initial and maintenance doses.
- [warfarin6](https://github.com/penpharmacy/warfarin6) - PWA tool for pharmacists to verify daily pill regimens and calculate exact pill counts.
- [warfarin-app](https://github.com/pharmacist-sabot/warfarin-app) - Fast regimen planner using Vue.js and Rust WebAssembly for intelligent dosing schedules.
- [warfarin-calculator](https://github.com/Paond55/warfarin-calculator) - Guideline-based PWA for calculating dose adjustments based on INR and bleeding symptoms.
- [Warfarin-Dose-Calculator](https://github.com/prajwal-17/warfarin-dose-calculator) - React + TypeScript app for anticoagulant dosing (Warfarin & Acitrom).
- [Warfarin-Dosage-Calculator](https://github.com/pzweuj/warfarin-dosage-calculator) - Next.js application implementing pharmacogenomic algorithms for initial dosage.
- [warfarin-planner](https://github.com/warfarin-net/warfarin-planner) - Dose-to-pill PWA with smart regimen sorting and dispensing calculations.

## 📱 Mobile Applications

Native or cross-platform mobile apps for patient empowerment, tracking, and reminders.

- [warfarin-management-app](https://github.com/amanjito/warfarin-management-app) - Comprehensive app with PT/INR tracking, reminders, and AI assistant (iOS/Android).

## 💻 Desktop Applications

Software for desktop operating systems suitable for offline use or deep data tracking.

- [INRTracker](https://github.com/starcalibre/inrtracker) - Native Windows (WPF) application for logging historical INR values and doses locally.

## 🧠 Machine Learning & Research

Projects leveraging ML, data analysis, and statistical modeling (e.g., IWPC dataset) for personalized medicine.

- [personalized-warfarin](https://github.com/cybergeist0/personalized-warfarin) - Jupyter Notebook predicting doses using clinical and genetic data (PharmGKB) via Random Forest.
- [pgx-warfarin-ml-portfolio](https://github.com/Uche-Stephen/pgx-warfarin-ml-portfolio) - Scikit-learn pipeline with SHAP explainability and a Streamlit web app.
- [warfit-learn](https://github.com/gianlucatruda/warfit-learn) - Python/scikit-learn project for predicting dosage with emphasis on reproducibility.
- [warfarin-machinelearning](https://github.com/karneslab/warfarin-machinelearning) - Comparison of ML models for predicting stable doses in diverse populations.
- [Warfarin_Dose_Prediction](https://github.com/farahyusri12/Warfarin_Dose_Prediction) - Colab notebook building a linear regression model from the IWPC dataset.
- [warfarin-dosing-ml-tool-](https://github.com/ifthekarhussain/warfarin-dosing-ml-tool-) - End-to-end ML pipeline with an interactive Gradio prediction tool.
